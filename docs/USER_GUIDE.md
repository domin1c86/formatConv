# User Guide

## Overview

FormatConv is a cross-platform format conversion tool that allows you to convert video, image, and audio files between different formats.

## Features

### File Selection
- **Single file selection**: Click "Browse Files" to select a single file
- **Multiple file selection**: Hold Ctrl (Windows/Linux) or Cmd (macOS) while clicking to select multiple files
- **Drag and drop**: Drag files from your file manager and drop them into the application

### Format Selection
- **Automatic detection**: The application automatically detects the type of your selected files
- **Format options**: Available formats are displayed based on file type
  - Video formats: MP4, MKV, MOV, AVI, WebM
  - Image formats: JPEG, PNG, WebP, TIFF, BMP
  - Audio formats: MP3, FLAC, WAV, AAC, OGG

### Conversion Settings
- **Quality options**:
  - Lossless: Maintains original quality (default)
  - Lossy: Reduces file size with quality loss
  - Quality slider: Adjust quality level (0-100%) for lossy conversion
- **Overwrite option**: Choose whether to overwrite existing output files

### Conversion Process
1. Select your input files
2. Choose the output format
3. Configure conversion settings
4. Click "Convert" to start
5. Monitor progress in real-time
6. Cancel conversion if needed

### Preview
- **File information**: View file details (resolution, duration, size) before conversion
- **Format information**: Hover over format cards to see format details
- **Result preview**: Open converted files with system default application

## Troubleshooting

### Common Issues
1. **Conversion fails**: Check if input file is corrupted or format is unsupported
2. **Slow conversion**: Large files take longer; consider using lossy compression
3. **Application crashes**: Ensure all dependencies are installed correctly

### Error Messages
- "File not found": Input file doesn't exist or is inaccessible
- "Unsupported format": File format is not supported
- "Conversion failed": Error during conversion process
- "Disk space insufficient": Not enough space for output file

## Keyboard Shortcuts
- `Ctrl+O` (Windows/Linux) / `Cmd+O` (macOS): Open file dialog
- `Ctrl+Enter` (Windows/Linux) / `Cmd+Enter` (macOS): Start conversion
- `Escape`: Cancel current operation

## System Requirements
- Windows 10+, macOS 10.15+, or Linux (Ubuntu 20.04+)
- 4GB RAM minimum
- 100MB free disk space
